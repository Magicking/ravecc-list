### Rave-cc importer

## Usage

```
Usage:
  ravecc-list [OPTIONS]

Application Options:
      --rpc-url=          Ethereum clients urls [$RPC_URL]
      --contract-address= ERC20 contracts addresses [$CONTRACT_ADDRESS]
      --private-key=      Base64URL encoded private keys [$PRIVATE_KEY]

Help Options:
  -h, --help              Show this help message
```
